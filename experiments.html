Experiments

This page demonstrates the proper use of template, picture, audio, video, canvas, source, and script tags.

SVG Experiment

1. <template> Element

The <template> tag holds HTML content that is not rendered by the browser by default.

2. <picture> & <source> Elements

The <picture> tag provides multiple <source> tags for "art direction." The browser will choose the best image.

3. <audio> & <source> Elements

The <audio> tag embeds a sound player. The controls attribute makes it visible.

4. <video> & <source> Elements

The <video> tag embeds a video player.

5. <canvas> Element

The <canvas> tag is a "drawing board".

Fallback text for browsers that do not support the canvas element.

6. <script> Element

The <script> tags are placed at the end of the <body>. One script performs the drawing on the canvas, and the other is a placeholder demonstrating how a third-party analytics script would be included.
